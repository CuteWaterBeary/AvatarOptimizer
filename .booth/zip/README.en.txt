Installation

a) Using Installer
  1. Import AvatarOptimizer-1.x.x-installer.unitypackage to your Unity project.
  2. Packages/com.anatawa12.avatar-optimizer is now added.

b) Using VCC
  1. Upgrade VCC to latest version
  2. Double-click add-repo.url to launch VCC
  3. Follow VCC to add the repository
  4. Add AvatarOptimizer to your project using VCC.

For other installation methods, see
 https://vpm.anatawa12.com/avatar-optimizer/en/#installation
